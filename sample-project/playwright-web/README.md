# Playwright Web Automation Project

## Overview
This project skeleton is designed to evaluate an AI assistant's ability to generate a Page Object Model (POM) based Playwright test suite for the SauceDemo web application.

## Prerequisites
* Node.js 20+

## Setup Steps
1. Install dependencies:
   ```bash
   npm install
   ```
2. Copy the `.env.example` to `.env`:
   ```bash
   cp .env.example .env
   ```
3. Install Playwright browsers:
   ```bash
   npx playwright install --with-deps
   ```

## Folder Structure
* `src/pages/` - Page Object Model classes
* `src/fixtures/` - Custom Playwright fixtures
* `src/utils/` - Global setup/teardown utilities
* `src/data/` - Test data files
* `tests/` - Spec files containing the test cases

*Note: The test scripts and POM classes are intended to be generated by the AI assistant during the evaluation phase.*

## Running Tests
Run all tests:
```bash
npm test
```
